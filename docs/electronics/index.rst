Electronics
===========

User Interface
----------------

.. elreq:: Provide LCD interface signals
   :id: ELR_001
   :links: SYR_001

   The hardware shall provide control and data lines for the LCD.

.. elreq:: Support HD44780 interface
   :id: ELR_002
   :links: SYR_001

   The hardware shall support HD44780-compatible signaling.
